# ne_toon_dl

네이버웹툰 무료 회차 자동 다운로드 (SJVA / FlaskFarm 플러그인)

## 주요 기능

- **설정** — 쿠키 등록, 다운로드 경로, 스케줄 주기, 알림 웹훅 설정
- **연재 목록** — 현재 연재중인 웹툰 전체 목록을 요일별로 보여주고, 원하는 작품을 체크해서 다운로드 스케줄에 바로 등록 (작품코드 수동 입력 불필요)
- **수동 다운로드** — 제목/URL/titleId 분석 후 받을 회차를 골라 즉시 다운로드
- **진행 상황** — 자동/수동 다운로드 실시간 진행률
- **다운로드 이력** — 작품·회차별 다운로드 상태 조회 및 정리
- 무료 회차만 자동 수집, 유료화 공지 감지, info.xml/cover.jpg 메타데이터 생성, ZIP 압축
