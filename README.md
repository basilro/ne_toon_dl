# 네이버웹툰 무료 회차 자동 다운로드 (ne_toon_dl)

SJVA / FlaskFarm 플러그인. 네이버웹툰 무료 회차를 스케줄에 따라 자동 다운로드한다.

---

## 설치

SJVA 플러그인 관리 → GitHub URL 입력:
```
https://github.com/basilro/ne_toon_dl
```

---

## 설정 항목

| 항목 | 기본값 | 설명 |
|---|---|---|
| **작품 목록** | (없음) | 한 줄에 하나 또는 `|` 구분. 제목 검색, 네이버웹툰 URL, title_id 숫자 모두 가능 |
| **쿠키 (JSON)** | (없음) | Cookie-Editor 익스텐션으로 export한 JSON. 네이버 로그인 쿠키 |
| **저장 경로** | (없음) | 다운로드 루트 폴더 경로 |
| **스케줄** | `0 9 * * *` | cron 형식(매일 09:00) 또는 분 단위 숫자(예: `180` = 3시간마다) |
| **시작 시 자동실행** | Off | 켜면 스케줄러 등록. 분 단위 숫자 interval이면 재시작 직후 1회 즉시 실행 |
| **유료화 공지 자동 다운** | Off | 매월 유료화 예정 작품을 공지 감지 시 자동 다운로드 |
| **유료화 저장 하위 폴더** | `완결` | 유료화 자동 다운 시 사용할 서브폴더명 |
| **ZIP 압축** | Off | 회차 다운 완료 후 이미지 폴더를 ZIP으로 묶고 원본 삭제 |
| **프록시 사용** | Off | warproxy 등 프록시 URL 설정 시 활성화 |
| **쿠키 만료 웹훅** | (없음) | 쿠키 만료 감지 시 Discord/Slack 알림 URL |
| **다운로드 웹훅** | (없음) | 완료 요약 + 경고/실패 알림 URL (같은 URL 사용) |

## 동작 방식

### 자동 다운로드
1. 작품 목록의 각 작품에서 **무료 회차** 목록을 확인
2. DB에 `completed` 기록이 없는 회차만 다운로드
3. 이미지 페이지당 **최대 3회 재시도** (타임아웃 대응)
4. 연속 8회 실패 시 해당 회차 중단 (다음 회차로 진행)
5. `partial` / `failed` 상태 회차는 **다음 실행 시 자동 재시도** (이전 폴더 삭제 후 처음부터)

### 회차 상태
| 상태 | 의미 |
|---|---|
| `completed` | 전체 페이지 다운로드 완료 — 이후 스킵 |
| `partial` | 일부 페이지만 성공 — 다음 실행 시 재시도 |
| `failed` | 페이지 다운로드 전체 실패 — 다음 실행 시 재시도 |
| `skipped_locked` | 유료(잠금) 회차 — 스킵 |

### Discord 알림
- **쿠키 만료**: 1회만 발송 (중복 방지), 다음 정상 실행 시 플래그 리셋
- **다운로드 완료 요약**: 받은 회차가 있을 때 발송
- **경고/실패 알림**: 작품 검색 실패, 불완전 다운로드 발생 시 발송

---

## 기술 사항

- **DB**: `ne_toon_dl.db` (플러그인 전용 SQLite, WAL 모드 자동 적용)
- **잠금**: 다른 플러그인(kaka_pe_dl 등)과 독립 실행 — 기다무 플러그인이 돌아도 블록 없음
- **UI**: 작업 중에도 작업내역 페이지 조회 가능 (WAL 덕분에 DB 락 없음)
