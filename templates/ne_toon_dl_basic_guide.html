📖 네이버웹툰 다운로드 — 사용 안내

빠른 시작

  1. 쿠키 주입 — comic.naver.com 로그인 → Cookie-Editor로 JSON export → 설정 > 일반 > 네이버 쿠키에 붙여넣기.
  2. 경로/작품 등록 — 다운로드 경로 지정 → 체크할 웹툰에 한 줄에 하나씩(제목·URL·titleId 숫자 모두 인식).
  3. 실행 — 설정 저장 → 쿠키 검증 → 즉시 실행 또는 자동 탭에서 스케줄러 등록.

탭 안내

탭용도
설정 > 일반 네이버 쿠키, 다운로드 경로, 체크할 웹툰, 회차 ZIP 압축, 유료화 공지 자동 다운, 프록시.
설정 > 알림 Discord/Slack 웹훅. 쿠키 만료 · 다운로드 완료 알림을 각각 켜고 끔.
설정 > 자동 스케줄러. Interval(분) 또는 Cron 형식. SJVA 시작 시 자동 실행도 옵션.
수동 다운로드 작품/회차 URL(또는 titleId)을 분석해 회차를 선택해서 즉석 다운. 이미 받은 회차는 비활성.
진행 상황 자동/수동 다운로드의 작품 · 회차 · 페이지 진행률 실시간 표시.
다운로드 이력 회차 단위 다운 이력. 상태 · 제목 검색 필터, 선택 삭제 지원.

├─ 작품제목/
│   ├─ info.xml · cover.jpg
│   ├─ 0001_회차제목/        (압축 Off)
│   │   ├─ 001.jpg · 002.jpg ...
│   └─ 0002_회차제목.zip      (압축 On)
└─ 완결/  (유료화 공지 자동 다운 — 폴더명은 설정에서 변경)
    └─ 작품제목/0001_회차제목/...

도구 버튼 (설정 탭 상단)

버튼동작
즉시 실행스케줄 무시하고 자동 다운로드 1회 실행.
쿠키 검증현재 쿠키 로그인 상태 확인.
공지 즉시 실행이번 달 유료화 전환 공지를 즉시 확인·다운(스케줄 무관).
메타 동기화작품 폴더 스캔해 info.xml/cover.jpg 누락분 생성.
압축 처리모든 회차 폴더를 zip으로 묶고 원본 삭제(이미 zip인 회차는 제외).
페이지수 부여회차 zip 파일명에 #페이지수 부여.
이력 초기화DB의 다운로드 이력만 비움(파일은 그대로).

꼭 알아둘 점

  • 무료 회차만 받습니다. 유료 회차는 대상이 아니며 카카오와 달리 이용권 차감 개념이 없습니다.
  • 유료화 공지 자동 다운을 켜면 매일 1회 공지를 체크해 이번 달 유료 전환 예정 작품의 무료 회차를 미리 받아둡니다(전환일이 지나면 자동 스킵).
  • 회차 ZIP 압축: On이면 정상 완료된 회차만 zip으로 묶고 원본 폴더 삭제. 부분 실패 회차는 그대로 폴더 유지.
  • 쿠키 만료: 필수 쿠키는 NID_AUT. 만료 감지 시 웹훅으로 1회 알림. 재로그인 → Cookie-Editor 재-export → 붙여넣기.
  • 이미지는 네이버 원본 jpg 그대로 저장 — 변환·재인코딩 없음.
이 플러그인은 본인이 정당하게 접근 권한을 가진 콘텐츠를 개인 보관 목적으로 내려받는 용도입니다. 네이버 이용약관과 저작권법을 준수해 사용하세요.
